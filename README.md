# How to run the jupyter lab

## Create and update image
```
$ cd docker
$ docker build -t kilimo_data_science .
```
Inside the folder you have a "requirements.txt" file where you can put the dependencies.  

## Run image and jupyter lab
```
$ cd ..
$ bash run_docker.sh
```
